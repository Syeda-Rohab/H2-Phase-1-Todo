# Create Hotel Task

Create a new hotel management task using the hotel-task-manager agent.

## Usage

This skill creates tasks for hotel operations including:
- Room housekeeping and cleaning
- Maintenance requests and repairs
- Guest services and requests
- Front desk operations
- Inventory management
- Staff assignments

## Prompt

Use the hotel-task-manager agent to create a new hotel task based on the user's requirements. Gather necessary details such as:
- Task type (housekeeping, maintenance, guest service, etc.)
- Priority level
- Room number or location (if applicable)
- Description
- Due date/time (if specified)
- Assigned staff (if specified)

Create the task with all relevant details and confirm successful creation.
