# Schedule Hotel Reminder

Schedule time-based reminders for hotel operations using the hotel-reminder-scheduler agent.

## Usage

This skill creates scheduled reminders for:
- Guest check-in/check-out notifications
- Housekeeping schedules
- Maintenance inspections
- Booking confirmations
- Wake-up calls
- Meeting room reservations
- Staff shift changes
- Special guest requests

## Prompt

Use the hotel-reminder-scheduler agent to create a scheduled reminder. Gather:
- Reminder type/purpose
- Date and time
- Room number or location (if applicable)
- Guest or staff name
- Notification method preference
- Any special instructions
- Recurrence pattern (if applicable)

Schedule the reminder and confirm with the scheduled time and details. For recurring reminders, specify the recurrence pattern clearly.
