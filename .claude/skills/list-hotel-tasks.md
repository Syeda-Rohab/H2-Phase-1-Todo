# List Hotel Tasks

View and filter hotel management tasks using the hotel-task-manager agent.

## Usage

This skill displays hotel tasks with filtering options for:
- Task status (pending, in-progress, completed)
- Task type (housekeeping, maintenance, guest services, etc.)
- Priority level
- Date range
- Room/location
- Assigned staff

## Prompt

Use the hotel-task-manager agent to retrieve and display hotel tasks based on the user's filter criteria. Present the tasks in a clear, organized format showing:
- Task ID
- Type and description
- Status
- Priority
- Location/room number
- Due date
- Assigned staff

If no filters are specified, show all pending and in-progress tasks.
