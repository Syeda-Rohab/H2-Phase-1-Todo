# List Hotel Reminders

View scheduled reminders for hotel operations using the hotel-reminder-scheduler agent.

## Usage

This skill displays scheduled reminders with filtering options for:
- Date range (today, this week, specific dates)
- Reminder type
- Room number or location
- Status (pending, completed, cancelled)
- Staff or guest name

## Prompt

Use the hotel-reminder-scheduler agent to retrieve and display scheduled reminders based on filter criteria. Present reminders in chronological order showing:
- Scheduled date and time
- Reminder type and description
- Associated room/location
- Guest or staff name
- Status
- Recurrence pattern (if applicable)

If no filters are specified, show all upcoming reminders for the next 24 hours.
