# Assign Hotel Task

Assign hotel tasks to staff members using the hotel-task-manager agent.

## Usage

This skill manages task assignments for:
- Housekeeping staff
- Maintenance team
- Front desk staff
- Guest services team
- Management

## Prompt

Use the hotel-task-manager agent to assign tasks to appropriate staff members. Gather:
- Task identifier
- Staff member name or role
- Any special instructions
- Priority or deadline adjustments

Assign the task and confirm the assignment. If assigning multiple tasks to one person or one task to multiple people, handle batch assignment efficiently.
