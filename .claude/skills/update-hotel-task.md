# Update Hotel Task

Update the status or details of a hotel management task using the hotel-task-manager agent.

## Usage

This skill updates hotel tasks including:
- Changing task status (pending → in-progress → completed)
- Updating priority
- Modifying description
- Reassigning staff
- Changing due dates
- Adding notes or comments

## Prompt

Use the hotel-task-manager agent to update the specified hotel task. Gather:
- Task identifier (ID, description, or room number)
- What needs to be updated
- New values

Update the task and confirm the changes. If the task cannot be found, help the user locate it by listing similar tasks.
