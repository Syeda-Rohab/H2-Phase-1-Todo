# Bulk Schedule Hotel Reminders

Schedule multiple hotel reminders at once using the hotel-reminder-scheduler agent.

## Usage

This skill efficiently creates multiple reminders for:
- Daily housekeeping schedules for multiple rooms
- Multiple guest check-ins/check-outs
- Recurring maintenance inspections
- Weekly staff meetings
- Multiple wake-up calls

## Prompt

Use the hotel-reminder-scheduler agent to create multiple reminders efficiently. Gather:
- List of reminders to create
- Common attributes (date, type, recurrence)
- Individual details for each reminder
- Any batch scheduling patterns

Process all reminders and provide a summary showing:
- Total reminders created
- Success/failure count
- List of created reminders
- Any errors or conflicts

Handle scheduling conflicts or errors gracefully and report them to the user.
